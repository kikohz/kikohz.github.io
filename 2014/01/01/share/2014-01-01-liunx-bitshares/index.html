liunx比特股挖矿

Posted on 2014-01-01 | In 原创 比特币 区块链

最近挖了挖PTS，今天刚好在liunx下找到了一篇教程，转载一下

已经出现多个PTS矿池了，推荐以下三个矿池：

http://pts.rpool.net

http://ptspool.com (鱼池)

http://ypool.net

挖矿软件有：jhProtominer_x32(推荐)、jhProtominer_x64(推荐)、Ptsminer、coyote_miner

经测，发现jhProtominer比ptsminer快了N倍，所以推荐用jhProtominer来挖！

Windows系统下直接下载jhProtominer解压后执行bat脚本即可，若是64位系统请下载jhProtominer_x64速度会比32位系统的要快！

Ubuntu linux下安装jhProtominer挖矿：

apt-get update
apt-get install -y git make g++ build-essential libminiupnpc-dev
apt-get install -y libboost-all-dev libdb++-dev libgmp-dev libssl-dev dos2unix
git clone https://github.com/Tydus/jhProtominer -b sph
cd jhProtominer/src/jhProtominer
make
cd jhProtominer/src/jhProtominer
./jhProtominer -o 112.124.13.238:28988 -u Pa3vV4VydA7khQLgpN3fojPiu8LTy3T4M8 -p x -m256

Centos Linux下安装jhProtominer挖矿：

yum -y install git gcc gcc-c++ libstdc++-devel
git clone https://github.com/Tydus/jhProtominer -b sph
cd jhProtominer/src/jhProtominer
make
cd jhProtominer/src/jhProtominer
./jhProtominer -o 112.124.13.238:28988 -u Pa3vV4VydA7khQLgpN3fojPiu8LTy3T4M8 -p x -m256

发现linux下的jhProtominer会经常自动退出，所以写了个脚本放到cron 5分钟检查一次，当然你也可以修改一下脚本放在后台5秒检测一次也可以，路径我都没修改了，要完善的自己修整一下吧！

vi /root/monminer.sh
#!/bin/sh
if [ ! -n "`ps -A | grep -v grep | grep jhProtominer`" ]; then
/root/jhProtominer/src/jhProtominer/jhProtominer -o 112.124.13.238:28988 -u Pa3vV4VydA7khQLgpN3fojPiu8LTy3T4M8 -p x -m256 &
fi

保存！

vi /etc/crontab
*/5 * * * * root /root/monminer.sh  >/dev/null 2>&1

保存！
