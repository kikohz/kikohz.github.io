从火车站的乘车经历想到的用户体验

发表于 2013-05-09 | 分类于 体验 杂文 | | 阅读次数:

转自http://blog.csdn.net/kevinkevin/archive/2011/06/15/6545576.aspx

这个故事应该大家都遇到过，的确很郁闷。

前两天出差，在苏州火车站乘车，离开车时间很近了我才赶到车站，手里拿着提前买好的车票，看着大厅上方悬挂的大屏幕，寻找我应该在几楼等车，在哪个 站台等车。大屏幕上显示每个车次信息：车次号/到站时刻/出发时刻/目的地/站台等。我现在知道我的车次号和出发时间，我就想知道在哪个站台等车。

不巧的是，这个屏幕并不能把全天的信息显示到一个屏幕上，它过大约10秒就刷新屏幕，显示另一个时间段的车次信息。8点到10点的信息已经过了，再显示 10点到12点，12点到14点，…, 我只好等，我的车次是9点钟的，我需要等它显示完一轮后重新回到我这个车次的时间段。眼睁睁看它从晚上到深夜，再回到凌晨，早上，OK, 到9点的时间段了，赶紧看，不料眼睛一花，还没找到我的车次呢，这一屏就过去了，我郁闷啊。

我就想知道我的车次，我的车次就在现在这个时间段，在屏幕下面看的其他人也一定是在未来一个或者两个小时乘车的，你给我们显示下午的或者晚上的车次信息， 而这与我们没有一毛钱的关系。为什么不把屏幕分成两部分，一部分是每隔半个小时刷新的，上面就显示目前2个小时之内的车次信息；另一部分还像原来的，隔 10秒种刷新一下，显示其他时间段的信息。这不就既满足了所有乘客的需求，又大大提高了那些需求最急的乘客的友好体验？

火车站的服务其实一直在提高，慢慢的变的越来越现代化，服务也越来越体贴。比如以前我坐火车，检票后走到站台上车，根据我的车票上的车厢号，去找相应的车 厢，但是我不知道这些车厢是怎么排的，是从前往后的呢，还是从后往前的？我需要连着看两个车厢，才知道这个顺序。现在我不需要了，站台上悬挂的液晶屏幕上 写了类似“1到9车厢的往后走，10到18车厢的往前走”，我立即就知道我该往哪边走了，要走多少距离。

用户体验满意度的提高就在这些小事情上，每一个进步都会让顾客感到温暖。国内很多地方的服务水平长久以来一直很差，稍微一些提高，我们就觉得很惊喜。但拿 国外某些地方比较，目前我们还有很大的差距。回想几年前我在德国出差，周末去柏林玩，从德雷斯顿坐火车到柏林，买票是从自动售票机上买的，选择好你能接受 的价位，车次类型，旅途耗用时间等，它会很智能的给你排出一个计划，中间转车越多越便宜。最能给我惊喜的是它的转车，我到一个小站下来，就是站里面的一个 站台，迈腿走两步，在站台的另一边等车，等约10分钟后，就可以换上另一趟车。你不需要换站台，更不需要出站，重新买票，问路等一系列麻烦事。

我认为检查一个火车站的服务水平，不能让一个经常出差的人去评论，而是让一个中等智力水平，初中毕业的，第一次到这个站的人来评判。如果这个人只凭自己可以顺利的入站，等车，上车，出站，不出错误，不走冤枉路，那么我们可以说，这个火车站的服务水平是合格的。
